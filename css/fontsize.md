# CSS font size

## ブラウザのデフォルトサイズは

100% = 1em = 1rem = 16px = 12pt

## Absolute

### px
pixel

### pt
point

16pxはどんなときも16px

## Relative

### Percent

親要素の何パーセントか。
50%であれば親要素の半分の大きさ

### em

1em = 100%
0.5em = 50%

### rem

ルート(html)に対する割合のサイズ
1em = 100%
0.5em = 50%　（デフォルト16pxであれば8px）

### vw vh

viewport width　ブラウザの幅に対する割合
viewport height　ブラウザの高さに対する割合

100vw = 100%

横幅320pxの時10vwは32px
